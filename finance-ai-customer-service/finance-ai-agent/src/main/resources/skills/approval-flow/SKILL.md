# 审批流程知识

## 概述
本技能提供企业财务审批流程相关知识，包括各类单据的审批步骤、所需材料和预计时间。

## 报销单审批流程

### 审批步骤
1. 员工提交报销单
2. 直属上级审批
3. 部门负责人审批（金额 > 2000 元时）
4. 财务审核
5. 出纳付款

### 所需材料
- 报销申请单（系统自动生成）
- 费用发票原件（电子发票需打印）
- 费用明细清单
- 相关审批单据（如出差审批单）
- 其他佐证材料（行程单、会议通知等）

### 预计时间
- 常规报销（≤ 2000 元）：3-5 个工作日
- 大额报销（> 2000 元）：5-7 个工作日
- 特殊报销（需特批）：7-10 个工作日

## 借款单审批流程

### 审批步骤
1. 员工提交借款申请
2. 直属上级审批
3. 部门负责人审批
4. 财务总监审批（金额 > 10000 元时）
5. 出纳放款

### 所需材料
- 借款申请单
- 借款用途说明
- 预算审批单（如适用）

### 预计时间
- 常规借款：3-5 个工作日
- 大额借款（> 10000 元）：5-10 个工作日

## 付款申请审批流程

### 审批步骤
1. 经办人提交付款申请
2. 部门负责人审批
3. 财务审核（核对合同、发票）
4. 财务总监审批（金额 > 50000 元时）
5. 出纳付款

### 所需材料
- 付款申请单
- 合同或协议复印件
- 供应商发票
- 验收单或确认函

### 预计时间
- 常规付款：5-7 个工作日
- 大额付款（> 50000 元）：7-15 个工作日

## 开票申请审批流程

### 审批步骤
1. 业务人员提交开票申请
2. 部门负责人审批
3. 财务审核开票信息
4. 开票员开具发票

### 所需材料
- 开票申请单
- 合同或订单信息
- 客户开票信息（名称、税号、地址、银行账号）

### 预计时间
- 常规开票：1-3 个工作日

## 常见问题

### 审批单被退回后如何处理？
1. 查看退回原因（系统通知或联系审批人）
2. 根据退回原因修改相关信息
3. 补充缺失的材料或附件
4. 重新提交审批

### 如何加急审批？
- 联系直属上级说明加急原因
- 在系统中标注"加急"标签
- 加急审批通常可在 1-2 个工作日内完成
